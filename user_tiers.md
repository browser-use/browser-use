# User Tiers

## Tier 1: Free
- Features: Basic access to features, limited usage.
- Ideal for: Users who want to try the service.

## Tier 2: Pro
- Price: $10/month
- Features: Unlimited access to features, priority support.
- Ideal for: Small businesses requiring more capability.

## Tier 3: Enterprise
- Price: $50/month
- Features: All Pro features + custom integrations and dedicated support.
- Ideal for: Large organizations with specific needs.
