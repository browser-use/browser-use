### Kimi Code [example](https://github.com/browser-use/browser-use/blob/main/examples/models/kimi_code.py)

[Kimi Code](https://www.kimi.com/code) is a coding-optimized AI model from Moonshot AI. The `kimi-for-coding` model excels at browser automation, coding tasks, and technical workflows with fast response times and strong reasoning capabilities.

<Info>
  Kimi-for-coding requires a Kimi Code subscription. Get your API key from [Kimi Code Console](https://www.kimi.com/code/console/api-keys).
</Info>

```python
from browser_use import Agent, ChatKimi
from dotenv import load_dotenv

load_dotenv()

# Initialize the model
llm = ChatKimi(model='kimi-for-coding')

# Create agent with the model
agent = Agent(
    task="Your task here",
    llm=llm
)
```

Required environment variables:

```bash .env
KIMI_API_KEY=
```

#### Model Features

- **kimi-for-coding**: Optimized for coding and browser automation tasks
  - Context length: 262,144 tokens
  - Supports reasoning
  - Supports image input (vision)
  - Supports video input

#### Configuration Options

```python
llm = ChatKimi(
    model='kimi-for-coding',
    api_key='your-api-key',  # Or set KIMI_API_KEY env var
    temperature=0.2,         # 0.0 to 1.0 (default: 0.2)
    max_completion_tokens=4096,  # Maximum tokens in response
)
```

<Warning>
  The Kimi-for-coding API has specific requirements:
  - `frequency_penalty` must be set to `0` (enforced automatically)
  - Uses "moonshot flavored JSON schema" which doesn't support `default` and `min_items` keywords (handled automatically)
  - Requires special headers to identify as a coding agent (handled automatically)
</Warning>
