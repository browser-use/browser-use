# คู่มือการกำหนดค่าระบบ ENV_CONFIG

ในเอกสารนี้เราจะพูดถึงการกำหนดค่าต่างๆ ของสภาพแวดล้อมในการพัฒนาหรือการใช้งานโปรแกรม

## การตั้งค่าพื้นฐาน

1. **DATABASE_URL**: ใช้สำหรับเชื่อมต่อกับฐานข้อมูล
2. **API_KEY**: ใช้สำหรับการเข้าถึง API ของเครื่องมือภายนอก
3. **ENVIRONMENT**: ระบุประเภทของสภาพแวดล้อม เช่น development, testing หรือ production

## การตั้งค่าขั้นสูง

1. **LOG_LEVEL**: กำหนดระดับการบันทึก เช่น DEBUG, INFO, WARN, ERROR
2. **CACHE_EXPIRY**: ระบุระยะเวลาห���ดอายุของแคช
3. **FEATURE_FLAGS**: ใช้ในการเปิดหรือปิดฟีเจอร์ในระบบ
